# YAI CLI Help Taxonomy v1

This note defines deterministic operator help taxonomy.

## Layer metadata

`layer` values are compatibility labels.
Primary ontology is:
- `core`
- `exec`
- `brain`
- `protocol`
- `platform`
- `support`

Historical layer labels (`boot/root/kernel/engine/mind`) may remain in metadata as migration aliases.
They are not primary ontology terms.
