- `control/` - control-plane schemas and authority surfaces
- `cli/` - CLI command schemas
- `vault/` - vault ABI headers and schema
- `graph/` - graph data contracts
- `providers/` - provider-facing schema contracts
- `engine/` - engine cortex schemas
